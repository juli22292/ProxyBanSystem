# ProxyBanSystem

Source-Code for Ultimate Ban System FREE

Dieselbe gebaute JAR kann auf Velocity, BungeeCord, Waterfall, Spigot, Paper, Folia, Leaf und Sponge eingesetzt werden.

Hinweise:
- Velocity: JAR in den Proxy-plugins-Ordner legen.
- BungeeCord/Waterfall: dieselbe JAR in den plugins-Ordner legen.
- Spigot/Paper/Folia/Leaf: dieselbe JAR in den Server-plugins-Ordner legen.
- Sponge: dieselbe JAR in den Sponge-plugins-Ordner legen.
- Die Plugin-Daten bleiben je nach Plattform im jeweiligen Plugin-Datenordner.
